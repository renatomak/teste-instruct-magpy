Sobre o projeto

Esta é uma API REST construída na fase de Teste Técnico para a vaga de Desenvolvedor(a) Python Júnior da Instruct. Esta aplicação se destina ao gerenciamento de projetos com foto em tecnologia. Auxiliando o usuário na pesquisa de pacotes válidos com versões válidas. Para casos em que o usuário não tenha uma versão definida, será atribuída a última versão do pacote. A aplicação utiliza a API pública da PyPi e tem como principal objetivo viabilizar a organização de projetos tecnológicos com pacotes consistentes e válidos.

Itens obrigatórios

  1. Sua API deve validar o projeto cadastrado: todos os pacotes informados devem estar cadastrados no PyPI. São validados o nome e a versão do pacote.
  2. Quando o pacote vem apenas com o nome, sua API deve assumir que é preciso usar a última versão publicada no PyPI.
  3. Se um dos pacotes informados não existir, ou uma das versões especificadas for inválida, um erro deve ser retornado.
  4. Deve ser possível visitar projetos previamente cadastrados, usando o nome na URL.
  5. Deve ser possível deletar projetos pelo nome.

Como testar a aplicação

  1. Crie o ambiente virtual para o projeto
    • python3 -m venv .venv && source .venv/bin/activate
  2. Instale as dependências;
    • python3 -m pip install -r requirements.txt
  3. Execute os testes;
    • Em um terminal, inicie o servidor
      • python manage.py runserver
    • E terminal diferente digite o comando:
      • python3 -m pytest
